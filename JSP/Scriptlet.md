
Scriptlet
======================================
* JSP 안에서 JAVA 언어를 사용할 수 있게 하는 요소 (<% / <%! / <%=)
  
  
---------------------------------------------------------------
<%! %>
---------------------------------------------------------------
* 변수(global), 함수, class 등을 선언하는 영역이다.

          <%!
          
          public void func(){
            System.out.println("func() 호출");	
          }	
            int gl_num = 0;
          %>
          
 --------------------------------------------------------------         
<% %>
---------------------------------------------------------------
* 구현부 영역이다. 제일 많이 활용한다. 

          <%
            func();

            int num = 0;

            gl_num++;
            num++;

            out.println("gl_num = " + gl_num);
            out.println("<br>");
            out.println("num = " + num);
          %>
      
--------------------------------------------------------------
<%= %>
--------------------------------------------------------------
* 결과값을 넣는 영역이다. 변수 및 html 태그 내에서도 활용할 수 있다.
          
          
          <%= String~ %>
          <p><%=123 %></p>
